# Food Delivery Sales Analytics Dashboard

## Project Overview
This project is an interactive Excel dashboard created to analyze food delivery sales data. The dashboard helps track important business metrics such as total sales, total orders, average rating, average order value, monthly sales trends, state-wise revenue, food type distribution, and quarterly performance.

This project was created as a self-learning / portfolio project to demonstrate Excel-based data analysis, Power Query data cleaning, PivotTable reporting, and dashboard creation skills.

## Tools Used
- Microsoft Excel
- Power Query
- PivotTables
- Pivot Charts
- Slicers
- Map Chart
- Excel Formulas

## Dataset
The dataset contains food delivery sales transaction records with details related to orders, restaurants, categories, states, ratings, sales amount, and food type.

Total records analyzed: 200,000+ transactions

## Key Steps Performed
- Cleaned and transformed raw sales data using Power Query
- Created calculated columns such as Food Type and Week Number
- Built PivotTables for sales, orders, ratings, and category-wise analysis
- Created KPI cards for important business metrics
- Designed an interactive dashboard using charts and slicers
- Analyzed monthly sales trends, state-wise revenue, food type distribution, and quarterly performance

## Dashboard Metrics
- Total Sales: ₹53.01M
- Average Rating: 4.34
- Total Orders: 197.43K
- Average Order Value: ₹268.51
- Food Type Distribution: 66% Veg and 34% Non-Veg
- Top-performing location: Bengaluru with ₹5.5M sales

## Dashboard Features
- Interactive slicers for restaurant, category, and time period
- KPI cards for quick business summary
- Monthly sales trend analysis
- State-wise revenue analysis using map chart
- Food type distribution analysis
- Quarterly performance comparison
